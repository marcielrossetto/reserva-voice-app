RossApp - Inteligência para Gastronomia

1. Isenção de Responsabilidade

O RossApp é uma plataforma de Business Intelligence e gestão de reservas. Ao utilizar este sistema, o estabelecimento (Restaurante) declara estar ciente de que:
  • A responsabilidade pela veracidade e obtenção de consentimento dos dados dos clientes finais é exclusiva do estabelecimento.
  • O RossApp fornece apenas o link para integração com o WhatsApp; o conteúdo das mensagens enviadas e a frequência do contato são de inteira responsabilidade do usuário (atendente/gerente).
  • Não nos responsabilizamos por bloqueios de contas de WhatsApp ou reclamações de spam.

2. Papéis e Responsabilidades (LGPD)

Para fins legais, o fluxo de dados no RossApp segue estas diretrizes:

  • RossApp (RossApp Inteligência): Atua como Operador. Nossa função é garantir que o banco de dados esteja seguro, isolado por empresa e disponível.
  • O Estabelecimento: Atua como Controlador. É o dono da relação comercial com o cliente e quem insere os dados no sistema.

3. Segurança e Isolamento SaaS

O RossApp utiliza arquitetura Multi-Tenant rodando em containers Docker, garantindo que:

  • Os dados da "Empresa A" são inacessíveis para a "Empresa B".
  • As senhas de funcionários são criptografadas (Hash) e nunca armazenadas em texto limpo.
  • Logs de Auditoria: O sistema registra qual usuário realizou cada ação, protegendo o dono do restaurante contra erros internos.

4. Instruções de Boas Práticas

Como manter seu restaurante seguro:
  1. Nunca compartilhe sua senha de acesso Master.
  2. Ao cadastrar um funcionário, defina o nível correto (Operacional ou Admin).
  3. Sempre peça autorização ao cliente antes de usar o número dele para fins de marketing (promoções).
  4. O RossApp mantém os dados para seu BI; use-os para melhorar o atendimento, não para incomodar o cliente.

5. Retenção e Exclusão

Os dados são de propriedade do estabelecimento. Em caso de encerramento de contrato, o RossApp manterá os dados por 90 dias para fins de conformidade fiscal e exportação, realizando a exclusão definitiva após este prazo.
